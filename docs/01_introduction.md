# 📘 Quipu 简介

> **The Effector for AI Agents.**

Quipu 是一个 **Markdown 原生的执行器**。

## 核心愿景

现代 AI 模型（LLM）具备强大的推理和生成能力，但在直接操作文件系统、执行复杂工程任务时，往往受限于“对话框”的交互形式。

Quipu 旨在填补这一鸿沟。它定义了一种简单的协议，将 Markdown 文件转变为**可执行的文档**。这使得 AI 生成的计划（Plan）不再仅仅是文本，而是可以直接作用于现实世界的脚本。

## 为什么选择 Quipu？

1.  **Markdown 原生**：直接执行 AI 最擅长生成的格式，无需复杂的 JSON/XML 转换。
2.  **幽灵引擎 (Phantom Engine)**：Quipu 不依赖脆弱的外部状态文件。它通过计算 Git Tree Hash 来实时感知环境，支持无痛的时间旅行和状态回滚。
3.  **零污染 (Zero Pollution)**：所有的状态计算都在影子索引中进行，不产生垃圾 Commit，不污染用户的暂存区。
4.  **安全可控**：内置路径逃逸检查、交互式 Diff 确认机制，确保 AI 不会“发疯”。

## 适用场景

*   **AI 辅助编程**：让 AI 直接修改代码库，并在出错时一键回滚。
*   **自动化运维**：编写包含解释性文字的可执行运维手册。
*   **教程演示**：创建既能阅读、又能直接运行的教学文档。
